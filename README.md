# 💻 [이름]의 개인 프로필 페이지

이 저장소는 GitHub Pages를 활용한 저의 개인 프로필 및 연구 포트폴리오 사이트 소스 코드입니다.
Jekyll 기반의 **Minimal Theme**를 사용하여 제작되었습니다.

🔗 **라이브 사이트 주소:** [https://계정아이디.github.io](https://계정아이디.github.io)

---

## 📂 프로젝트 구조

* `_config.yml`: 사이트의 전반적인 설정(제목, 테마, 소셜 링크 등)을 담당합니다.
* `index.md`: 웹사이트의 메인 화면에 표시되는 실제 내용(자기소개, 논문 목록 등)입니다.
* `assets/`: 이미지, PDF 파일 등 정적 리소스가 저장되는 폴더입니다.

---

## 🛠 수정 및 업데이트 방법

1. **정보 수정:** `_config.yml`에서 이름과 소셜 미디어 아이디를 변경할 수 있습니다.
2. **내용 업데이트:** `index.md` 파일의 마크다운 내용을 수정하면 실시간으로 웹사이트에 반영됩니다.
3. **논문 추가:** `index.md` 하단의 `Publications` 섹션에 새 항목을 추가하세요.

---

## 🚀 로컬에서 실행하기 (개발자용)

이 사이트를 내 컴퓨터에서 미리 보려면 Ruby와 Jekyll이 설치되어 있어야 합니다.

```bash
# 의존성 설치
bundle install

# 로컬 서버 실행
bundle exec jekyll serve
